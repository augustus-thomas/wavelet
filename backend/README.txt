This project requires Python 3 and the ttkThemes/Tkinter installed.

Run main.py for simulation. To change between UI and data collection check the top of main.py file.

To change starting parameters edit variableSettings.py

e, f, q are chemical parameters delta is the activator threshold (any activator below this value is considered 'dead') phi is the maximum light in the reaction Du is the diffusion constant in the activator Diffusion for inhibitor is assumed to be 0. N is side length on the simulation (NxN) uExcited is the variable that the activator starts with
